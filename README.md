# Miaudota - Aplicativo de Adoção Responsável de Animais

## Sobre o Projeto

O Miaudota é um aplicativo mobile desenvolvido como parte da disciplina Hands On Work X (HOW X), com o objetivo de promover a adoção responsável de animais por meio da tecnologia.

A proposta surgiu a partir da identificação de dificuldades enfrentadas por adotantes, protetores independentes e instituições de acolhimento na divulgação de animais disponíveis para adoção e na comunicação entre as partes envolvidas.

## Objetivo

Desenvolver uma solução tecnológica capaz de facilitar o processo de adoção responsável, ampliando a visibilidade dos animais disponíveis e aproximando adotantes, protetores independentes, ONGs e abrigos.


## Funcionalidades do MVP

- Cadastro de animais para adoção;
- Listagem de pets disponíveis;
- Filtros de busca;
- Visualização de detalhes dos animais;
- Cadastro e autenticação de usuários;
- Comunicação direta via WhatsApp.

## Metodologia

## Como executar o projeto

### Backend
1. Acesse a pasta `MiaudotaAPI`
2. Instale as dependências:
```bash
npm install
```

3. Execute:
```bash
npm start
```


### Mobile
1. Acesse a pasta `miaudotaApp`
2. Instale as dependências

```bash
flutter pub get
```

3. Execute a aplicação

```bash
flutter run
```

## Arquivo executável

O projeto inclui uma versão em APK, disponibilizada para validação e testes do MVP, permitindo a execução da aplicação em dispositivos Android.
